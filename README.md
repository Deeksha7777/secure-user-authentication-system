# Secure User Authentication System

A full-stack user authentication web application built using:

- Java
- Spring Boot
- MySQL
- Thymeleaf
- Bootstrap

## Features

- User Registration
- User Login
- Duplicate username validation
- Clean responsive UI
- MySQL database integration

## Tech Stack

Backend:
- Spring Boot
- Spring Data JPA
- MySQL

Frontend:
- HTML
- CSS
- Bootstrap

## How to Run

1. Clone the repository
2. Configure MySQL in `application.properties`
3. Run `UserauthApplication.java`
4. Open:
   - http://localhost:8080/register-page
   - http://localhost:8080/login-page
